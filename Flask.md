# Flask

* 경량형 프레임워크
* response 확인 가능한 micro web framework(까페 창업시 프렌차이즈 창업과 같음)
* 웹서비스 만들때 적합하지 않음



## 설치

```git
pip install flask
```



## 실행

* c9에서

  ```git
  flask run --port=8080 --host=0.0.0.0
  ```

  flask에 8080으로 보내면 flask 자체에서 80번 포트로 외부에 보내겠다.

* local에서

  ```git
  flask run
  ```

  * local에서는 해당 컴퓨터에서만 접속 가능
